dkim_hashalg_t

[back to index]

DESCRIPTION
Purpose A dkim_hashalg_t is a value from an enumerated list of the possible hash algorithms available. It is passed in the form of an argument type as a parameter to dkim_sign() when selecting a hash algorithm to be used when preparing a message body for signing. It need not be passed to dkim_verify() as that function will determine the hash algorithm for verification based on the signature on an arriving message.
Possible Values
  • DKIM_HASHALG_SHA1 -- defined in RFC 6376
  • DKIM_HASHALG_SHA256 -- defined in RFC 6376
